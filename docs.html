Crystal CSS Framework

一个现代化的CSS框架，采用毛玻璃效果和简约设计理念，提供丰富的UI组件和响应式布局系统。

特色功能： 毛玻璃效果、暗黑模式支持、响应式设计、流畅动画、丰富组件

主要特性

毛玻璃效果

内置多种毛玻璃效果，营造现代化的视觉体验

响应式设计

完全响应式布局，适配各种屏幕尺寸

暗黑模式

原生支持暗黑模式，自动适配系统主题

流畅动画

精心设计的过渡动画，提升用户体验

安装使用

国际CDN

国际用户推荐使用：

HTML
<link rel="stylesheet" href="https://cdn.yxpil.com/crystal.css"> <script src="https://cdn.yxpil.com/crystal.js"></script>

国内CDN

国内用户推荐使用，速度更快：

HTML
<link rel="stylesheet" href="https://xianmoxukong.top/cdn/crystal.css"> <script src="https://xianmoxukong.top/cdn/crystal.js"></script>

本地引入

下载文件后本地引入：

HTML
<!-- CSS文件 --> <link rel="stylesheet" href="Crystal.css"> <!-- JavaScript文件（可选） --> <script src="Crystal.js"></script> <!-- Font Awesome图标（推荐） --> <link rel="stylesheet" href="fontawesome-free-7.0.1-web/css/all.min.css">

GitHub 克隆

从 GitHub 仓库获取最新版本和源码：

BASH
git clone https://github.com/Yxpillow/crystalcss.git
说明： 推荐优先使用CDN引入方式，方便快捷。开发者可以通过GitHub克隆获取源码进行二次开发。

暗黑模式

Crystal CSS 原生支持暗黑模式，可以通过JavaScript API或开关按钮进行切换。

JavaScript API

// 切换暗黑模式 Crystal.toggleDarkMode(); // 设置暗黑模式 Crystal.DarkMode.setDark(true); // 开启 Crystal.DarkMode.setDark(false); // 关闭 // 获取当前模式 const isDark = Crystal.DarkMode.isDark();

开关按钮

使用 data-switch="darkmode" 属性创建暗黑模式开关：

<label class="switch" data-switch="darkmode"> <input type="checkbox" class="dark-mode-toggle"> <span class="switch-slider"></span> </label>

自定义暗黑模式样式

/* 亮色模式样式 */ .my-component { background: #ffffff; color: #333333; } /* 暗黑模式样式 */ [data-theme="dark"] .my-component { background: #2d2d2d; color: #ffffff; }
